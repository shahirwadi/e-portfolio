View Report

DETAIL

My team which is Code Bois have done the Comparative Evaluation of Streaming Service Websites using Google Lighthouse.

Based on an article published by Forbes called “The Best Streaming Services For 2024”, we have chosen Netflix, Hulu, Amazon Prime Video, Disney +, Apple TV +, and Paramount + websites to undergo a Web Performance Audit. To audit, we will be using Google Lighthouse which is an open-source automated audit tool. It will assist in improving the quality of the website by generating performance, accessibility, best practices, and Search Engine Optimization (SEO) analysis.

SUMMARY

Working in a group has had a real impact on me, during the report-creation process, I learned new information about SEO, best practices, accessibility, performance, and Progressive Web Apps (PWAs), which are used to assess websites.

Additionally, I developed knowledge using Google Lighthouse, a tool for evaluating the functionality of websites. We were then able to identify areas for development and draw lessons from the actual case.
